## 🔧 Customization Guide

### To Change Primary Color
Edit frontend/css/styles.css line ~11:
--accent: #FF8C42;           /* Change this hex code */
--accent-glow: rgba(255, 140, 66, 0.5);

### To Adjust Animation Speed
Edit any animation duration:
animation: float 4s ease-in-out infinite;
           └─ Change \"4s\" to your desired duration

### To Disable Glow Effects
Set glow opacity to 0:
--accent-glow: rgba(255, 140, 66, 0);
--secondary-glow: rgba(255, 71, 87, 0);

### To Change Glow Intensity
Modify the opacity values (0.5 = 50%, 0.8 = 80%):
--accent-glow-strong: rgba(255, 140, 66, 0.8);
